# Cracked vs. Non-Cracked Surface Detection
An end-to-end deep learning project focused on computer vision, designed to automatically detect structural cracks on surfaces. This repository explores a wide variety of neural network architectures—ranging from baseline Convolutional Neural Networks (CNNs) to Multi-Directional RNNs and Transfer Learning approaches—to determine the most effective method for surface anomaly detection.

# 📊 Dataset
The project utilizes the [Cracked/Non-Cracked Surface Dataset](https://www.kaggle.com/datasets/geadalfa/cracked-non-cracked-surface-datasets) from Kaggle.

- The data undergoes rigorous Exploratory Data Analysis (EDA), including grayscale conversion and Sobel Edge Magnitude visualization, to highlight structural anomalies before being fed into the models.
